*** How to run this code ***

1. Make sure you are in the virtual environment (in terminal, source .venv/bin/activate)
2. cd to the project folder containing server.py (cd presidio_analyzer-2.2.360-py3-none-any)
3. To run, in terminal, type python server.py
4. This should start the server, click on the local address to open the webpage
5. Type anything into the input box, debugging statements are printed to console if needed
